# API5 – Broken Function Level Authorization

## 📌 Vulnerability Type
OWASP API5: Broken Function Level Authorization

## 🎯 Description
Normal user accessing admin-only endpoint.

## 🔍 Vulnerable Endpoint
GET /admin/dashboard

## ✅ Result
Unauthorized admin access.
